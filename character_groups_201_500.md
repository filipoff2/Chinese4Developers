# Chinese Character Groups (201–500 Range)
Grouped by Shared Components — Chinese → English

This file extends beyond the basic characters.

so shared characters make memorization easier.

## 依 / 亻 — Dependence, relation

Core idea: **reliance, relation between things**

- 依赖 — dependency  
- 依存 — depend on  
- 依据 — basis  
- 相关 — related  
- 关联 — association  
- 关系 — relation  

Example wording:
- 服务 依赖 数据库 — service depends on database
